# DW EXA1 — Landing Page en React y Vue ☕

Examen: desarrollo y despliegue de una misma landing page (cafetería
**Café Aroma**) en dos tecnologías, **totalmente basada en componentes**.

## 📂 Contenido del repositorio
- [`react-cafeteria/`](react-cafeteria/) — Landing page desarrollada con **React + Vite**.
- [`GUIA.md`](GUIA.md) — Guía completa: ejecución, despliegue, GitHub, guion del video y comparativa técnica.

> 🔗 La versión en **Vue 3 + Vite** (mismo diseño) está en un repositorio aparte:
> https://github.com/RonnyAreMC/DBEXA12

## 🚀 Ejecutar el proyecto
```bash
cd react-cafeteria
npm install
npm run dev
```

## ✨ Características (en ambas versiones)
- Diseño visual organizado y **responsive**.
- Arquitectura **basada en componentes** (cada sección y cada campo del formulario es un componente reutilizable).
- **Formulario funcional** con validaciones básicas en el frontend.
- Navegación clara entre secciones.
